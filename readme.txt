This is the project to store statistic for playing air battle simulator "World of warplanes"

This tool may be useful to end players because official statistic does not allow to see effectivity of its game based on win. rate


Project developing with following tools:
- Visual studio community 2017
- Sqlite database enginie
- Dbdesigner.net

Project structure 
- doc / 
		Any materials such as initial task or user manual
- db /
		Database, 
		Initial scripts of structure creation, 
		Initial scripts of constant data filling.
- app-clinet /
		Windows client application
